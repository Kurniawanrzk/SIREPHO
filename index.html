SIREPHO INSTALLER 1.0 ver

Aplikasi manajemen resepsionis berbasis java

Instruksi Instalasi SIREPHO
  1. Persiapan Lingkungan

    Pastikan Anda memiliki Java Runtime Environment (JRE) atau Java Development Kit (JDK) yang dibutuhkan untuk menjalankan aplikasi

  2. Unduh Aplikasi SIREPHO

    Klik tombol "Unduh SIREPHO" untuk mengunduh file aplikasi. Simpan file di lokasi yang mudah diakses.

  3. Instalasi dan Pengaturan

    Periksa apakah JRE atau JDK sudah terinstal dengan menjalankan perintah java -version di command prompt atau terminal.
